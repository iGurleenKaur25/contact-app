# Contact Management App (MERN)

A simple contact management web application built using the MERN stack.

## Features
- Add contact
- View contacts
- MongoDB database
- REST API using Express

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
